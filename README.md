# お手伝いクエスト（homework-quest）

子どもの宿題・お手伝いを、本人がチェックして進める試作アプリ。
達成すると **ポイント → レベルアップ／バッジ** で達成感が出て、頑張りは
**カレンダー・連続日数（ストリーク）・グラフ** で見える化します。

- 一人用。子どもが自分で操作（タブレット/スマホ想定）、親はタスクを登録
  - きょうだいで使う場合は、それぞれの端末/フォルダで別々に動かす想定
- チェックしたら即ポイント確定（承認フローなし）
- 定番ルーティン（毎週この曜日）＋スポット（この日だけ）の両対応

## 使い方

```bash
npm install
npm run dev        # または npm start
```

起動後、ブラウザで <http://localhost:3100> を開く。ポート変更は `PORT=4000 npm start`。

## スマホ・タブレットから使う

サーバを起動すると、ターミナルにスマホ用のURLが表示される（例 `http://192.168.0.29:3100`）。

1. スマホをPCと**同じWi-Fi**につなぐ
2. スマホのブラウザで、表示された `http://<PCのIP>:3100` を開く
3. ブラウザの「ホーム画面に追加」でアプリのように使える

PCのIPは起動時のログに出る（変わることがあるので、つながらない時はログを確認）。

### つながらない時のチェック
- **同じWi-Fiか**：スマホとPCが同一ネットワークにいるか
- **VPN**：PCでVPN（NordVPN等）が有効だと、ローカルネットワークがブロックされることがある。
  VPN設定で「ローカルネットワークを許可」を有効にするか、一時的にVPNを切る
- **ファイアウォール**：初回は「アクセスを許可しますか？」が出たら「許可」を選ぶ
- **PCがスリープ**するとサーバが止まる。使う間はPCを起動しておく

## 画面

- **今日**（`/index.html`）… 今日のタスク。タップで達成・ポイント加算、レベルバー、バッジ、お祝い演出
- **記録**（`/record.html`）… ストリーク・週間/月間集計・棒グラフ・カレンダー
- **設定**（`/parent.html`）… 定番/スポットタスクの追加・削除、名前・色の編集

## データ

`data/*.json` にファイル保存（初回起動時にサンプルを自動シード）。

- `children.json` … 子ども（名前・色・XP・レベル・バッジ）
- `tasks.json` … タスク定義（定番/スポット）
- `completions.json` … 達成ログ（見える化・集計の元データ。XP等はここから再計算）
- `config.json` … レベル閾値・バッジ定義（編集して調整可能）

データを初期状態に戻したいときは `data/` フォルダごと削除して再起動。

## 技術メモ

- Node.js + Express、フロントはビルド不要のバニラJS（マルチページ）
- 依存は `express` のみ
- XP/レベル/バッジは `completions.json` を「正」として毎回再計算するため、
  加算・取り消しでズレが起きない設計
